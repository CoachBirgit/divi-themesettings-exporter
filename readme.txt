Export/Import your Divi Themesettings with no mess

IMPORTANT: Needs a active install of the Premium Theme Divi or a Divi childtheme!

== Description ==

Go to "Tools > Backup Options" to export/import theme settings

== Installation ==


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Tools > Backup Options" to export/import theme settings
